Установка

Установка репозитория

  1. Скачиваем дополнение: repository.lanet.ua

  2. Запускаем плеер

  3. Переходим:

    1. Система->Настройки

    2. Дополнения

    3. Установить из файла ZIP

    4. Ищем скачаный файл и выбираем его

    5. Если все сделано правильно, появится сообщение "Lanet Network Add-ons for Kodi, дополнение включено"

Установка клиента PVR

  1. После установки репозитория, выбираем Загрузить дополнения

  2. Свежеустановленный репозиторий Lanet Network Add-ons for Kodi

  3. Клиенты PVR

  4. Выбираем подходящую версию клиента

  5. Установить

  6. Если все сделано правильно, появится сообщение ...дополнение включено

Включение функционала ТВ в плеере*

*если ранее не использовалось

  1. Система->Настройки

  2. ТВ

  3. Общие->вкл.

  4. После выхода на главный экран плеера, в меню появится новый пункт ТВ

  5. Немного скриншотов работающиего дополнения
